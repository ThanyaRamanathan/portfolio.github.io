Image Translation and Unsupervised Domain Adaptation

Project

This project focuses on the intersection of image translation and Unsupervised Domain Adaptation (UDA). By leveraging the power of GANs, the project aims to bridge the domain gap, enabling the trained model to generalize effectively in a target domain despite lacking labeled data. The integration of UDA principles ensures that the adapted model can achieve improved accuracy and robustness when applied to real-world scenarios with domain shifts.

Next Point of Recommendation using Large Language Models

Master's Thesis Project

The recommendation model synergizes the GETNext Algorithm with the Knowledge Reasoning and Generation (KRG) stage from the Open-World Knowledge Augmented Recommendation (KAR) framework, leveraging the Stanford Alpaca model for generating contextually rich textual descriptions of user preferences. These descriptions, derived from tailored prompts based on user attributes and behaviors, provide nuanced insights that go beyond traditional numerical or categorical data. Semantic embeddings, generated using pre-trained models like Sentence-Transformer, convert these descriptions into dense vectors, enriching user representations within the GETNext Algorithm. This approach combines historical interactions with inferred interests to deliver highly personalized recommendations, marking a significant advancement in recommendation systems by seamlessly integrating numerical and textual data for enhanced accuracy and user experience.

MDMTSP using Genetic Algorithm and Reinforcement Learning

Research Paper and Bachelor's Thesis Project

This research addresses the closed path multi depot MTSP problem, solved using Genetic Algorithm. The genetic algorithm, which is based on natural selection, is a technique for resolving both constrained and unconstrained optimization problems. The Genetic algorithm is especially useful in solving Np-hard problems. Using the elitist selection operator, 2-opt mutation and OX crossover, and further optimized with the help of Reinforcement learning with to get a feasible solution.

Publication Records Analysis and Processing

Project

The project encompasses two key phases: text analysis and search engine development. In the first phase, three distinct datasets were analyzed: initial chapters of poorly rated fanfiction from Archive of Our Own (AO3), satirical news articles from The Onion, and spell descriptions from Dungeons and Dragons (DND). Natural language processing techniques, including Treebank tokenization, stemming, and sentence tagging, were applied to uncover linguistic patterns and thematic insights unique to each domain, such as narrative styles, satirical features, and gaming-specific language. The second phase involved developing a search engine using the DBLP dataset, enabling efficient querying of scholarly literature by author, title, publication year, and journal. Advanced algorithms ensured precise and relevant results, facilitating exploration across diverse scientific domains.
